Setup Eclipse Workspace

To setup the eclipse workspace do the following:

  1. Open Workspace:  Open the workspace containung the Coreframework with eclipse.
  2. Import Projects:  If the projects are not listed in the project explorer, use "File >> Import..." to import the projects.
  3. Create User Library: 
    • Go to "Window >> Preferences >> Java >> Build Path >> User Libraries"
    • Create a new library and name it "CFW"
    • Add the jar files from "CoreFramework/lib" to the new library. The project should now be able to build without errors.

Run

To run a project open the applications Main-class and hit the run button in Eclipse.

Export to Standalone Application

To export your application do the following:

  1. Right click the project folder of your application and choose "Export...".
  2. Choose Runnable Jar File and click next.
  3. Select the launch configuration of your application.
  4. Choose a path for the resulting jar file.
  5. Choose option "Extract required libraries into generated jar"
  6. Copy the required files and folders, in general these are:
    • /config
    • /recources
    • /scripts/start.bat
    • /scripts/stop.bat
